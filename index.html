Core Memory Music

Great music, memorable performances

Core Memory Music is a performing arts venture based in southern Rhode Island. We organize and host concerts in a modern home near a scenic mill pond. The performance space is a spacious living room with comfortable seating and a Steinway concert grand piano. Accomplished young musicians from around the world come here to perform great music in a relaxed, informal setting.
